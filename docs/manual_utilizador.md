# Manual do Utilizador

## Aceder à aplicação
Após iniciar a aplicação, a API fica disponível em:
http://localhost:8080

## Swagger
A documentação interativa pode ser acedida em:
http://localhost:8080/swagger-ui.html

## Operações principais
- Criar item
- Listar itens
- Atualizar item
- Registar movimentos de stock
- Consultar stock atual
- Consultar itens abaixo do stock mínimo

Todas as operações podem ser testadas diretamente no Swagger.
